# 中国农业绿色转型监测大屏 —— 需求文档

## 文档信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 中国农业「减量增效」绿色转型监测大屏 |
| 文档类型 | 产品需求文档（PRD） |
| 核心价值 | 展示政策对农业绿色转型的驱动作用 |
| 数据来源 | 国家统计局分省年度数据（2021-2024） |
| 政策依据 | 2015零增长→2017五大行动→2021绿色规划→2022减量化→2024一号文件 |

---

## 一、项目背景

2015年以来，国家持续出台化肥农药减量政策。2022年《到2025年化肥减量化行动方案》设定了量化减量目标。2024年中央一号文件明确提出"推进化肥农药减量增效"。

当前需要一个可视化大屏，不是"验证政策效果"，而是**展示政策如何驱动了农业绿色转型**——哪些省份响应了政策、响应程度如何、下一步应在哪里加大推动力度。

## 三、非功能需求

| 需求编号 | 需求描述 | 指标 |
|---------|---------|------|
| NF01 | 首屏加载时间 | ≤3秒 |
| NF02 | 地图交互响应 | ≤200ms |

### 3.2 视觉需求

| 需求编号 | 需求描述 |
|---------|---------|
| NF05 | 全屏无滚动，适配1920×1080和1366×768分辨率 |

| NF09 | 所有图表标签不被裁切，不出现横向滚动 |

| 需求编号 | 需求描述 |
|---------|---------|
| NF10 | 数据从本地CSV文件读取，无需后端服务 |
| NF11 | 数据预处理为JSON格式嵌入HTML |
| NF12 | 支持数据更新后直接替换JSON重新加载 |

## 六、数据流

```
CSV原始数据（分省数据目录）
        │
        ▼
Python预处理脚本
  ├── 读取18类CSV文件
  ├── 计算7项衍生指标
  ├── 计算3维度政策响应度
  ├── 合并为统一JSON
  └── 输出嵌入HTML
        │
        ▼
单个HTML文件
  ├── Vue 3 数据绑定
  ├── ECharts 图表渲染
  ├── 全屏无滚动布局
  └── 浏览器直接打开
```
