# 中国农业绿色转型监测大屏 —— 需求文档（实现版）

## 文档信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 中国农业「减量增效」绿色转型监测大屏 |
| 文档类型 | 产品需求文档（PRD）—— 与实际实现同步 |
| 核心价值 | 展示政策对农业绿色转型的驱动作用 |
| 数据来源 | 国家统计局分省年度数据（**2016-2024**） |

---

## 一、项目背景

2015年以来，国家持续出台化肥农药减量政策。2022年《到2025年化肥减量化行动方案》设定了量化减量目标。2024年中央一号文件明确提出"推进化肥农药减量增效"。

本大屏的核心叙事是：**展示政策如何驱动了农业绿色转型**——以各省化肥施用量相对于全国减量速度的"响应度"为核心指标，通过 6 个功能模块 + 1 条政策时间轴，呈现投入减量、产出保障、替代机制、经济效益、区域效率、行动建议的全链条监测体系。

**实现要点**：

## 七、核心指标计算公式

| 指标 | 公式 | 使用模块 |
|------|------|---------|
| **响应度** | `(省份2016→Y年化肥减量率) ÷ (全国2016→Y年化肥减量率)`，截断至 [0, 3] | 模块1、模块6 |
| **响应等级** | ≥1.5 高度 / 1.0~1.5 正常 / 0.5~1.0 弱响应 / ≤0.5 未响应 / ≈0 未知 | 全部模块 |
| **亩均施肥量** | `Σfertilizer / ΣplantArea × 666.67` (kg/亩) | 模块4 |
| **亩均产值** | `ΣagriOutput / ΣplantArea × 6666.67` (元/亩) | 模块4 |
| **效率（吨粮/吨肥）** | `grain / fertilizer` | 模块5 |
| **区域六维评分** | 6项指标分别 0-100 标准化后取各省均值 | 模块5 |
