### 数据架构
- MySQL 8.0.40，`agriculture_green`，12,284 条记录
- 表：`indicator_data` (province, year, category, indicator, value)
- 数据源：`分省数据/数据/` 14类 + `全国数据/全国数据/` 3类
- 后端：Express `localhost:3001`

### 常用命令
```bash
# 数据导入
cd dashboard/backend && node import_data.js

# 启动后端
cd dashboard/backend && node server.js

# 启动前端
cd dashboard/frontend && npm run dev

# 测试 API
curl http://localhost:3001/api/dashboard
```
